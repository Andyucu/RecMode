RecMode User Guide

Version 0.9.27-beta · Windows screen recorder · Last updated July 2026

RecMode captures your desktop, windows, or regions, encodes with hardware or software ffmpeg, and saves shareable video files — with optional audio, webcam overlay, annotations, and automation.

Overview

RecMode is a portable-first Windows application. After you extract or install it, recordings and settings stay in folders next to the executable (portable mode) or in your chosen install location. No account, no cloud upload, and no telemetry.

┌─────────────┐ ┌──────────────┐ ┌─────────────┐ ┌──────────┐ │ Capture │ ──► │ GPU pipeline │ ──► │ ffmpeg │ ──► │ Output │ │ WGC / DXGI │ │ NV12 + audio │ │ HW or SW enc│ │ MP4/MKV… │ └─────────────┘ └──────────────┘ └─────────────┘ └──────────┘ ▲ ▲ │ └── Brightness, region crop, webcam PIP └── Screen / Window / Region / All Displays

Main areas of the app:

  • Record — choose what to capture, adjust quality and audio, start/stop.
  • Library — browse saved videos and screenshots.
  • Schedule — set timed recordings that fire while RecMode is running.
  • Settings — defaults, appearance, hotkeys, performance, and startup.

Installation

Portable (recommended for USB or no-install use)

  1. Download RecMode-*-portable-win-x64.zip and extract to any folder.
  2. Run RecMode.exe.
  3. The presence of portable.marker tells RecMode to keep all data inside this folder.

Installer

Run RecMode-win-Setup.exe for a guided install, or deploy RecMode-win.msi in managed environments. Both include bundled ffmpeg and third-party license notices.

Custom location (Setup):

RecMode-win-Setup.exe --installto "D:\Apps\RecMode"

Custom location (MSI):

msiexec /i RecMode-win.msi VELOPACK_INSTALLDIR="D:\Apps\RecMode"

Start with Windows

Enable Settings → General → Start with Windows to launch RecMode minimized to the tray at sign-in (RecMode.exe --tray).

Basic recording steps

  1. Select a source tile (Screen, Window, or Region).
  2. Configure the target (display, window, or region).
  3. Choose a Profile or adjust encoder, format, frame rate, and quality.
  4. Set Audio and optional Webcam overlay.
  5. Click Record or press F9.
  6. When finished, click Stop or press F9 again.

If countdown is enabled, a full-screen 3-2-1 overlay appears before recording starts (Esc cancels). CLI --record skips the countdown.

The status bar shows elapsed time, live stats (fps, bitrate, size), and free space on the drive where recordings are saved.

Capture sources

Screen

Records an entire monitor. Choose the display from the Display dropdown in the Video card. If you have two or more monitors, an All Displays entry composites every monitor into one virtual-desktop-sized recording.

Window

Records one application window. After clicking the Window tile:

  • Dropdown — lists open top-level windows (title-less overlays are excluded).
  • Manual Pick — dims the screen; move the mouse to highlight a window, then click to select it.
  • Follow selected window — before capture starts, RecMode re-resolves the window by handle, then process+title, so apps that recreate their window can still be recorded without repicking.

While attached to a window handle, Windows Graphics Capture follows normal move and resize.

Region

Records a rectangle on a monitor. Click the Region tile to open the region picker:

  • Drag to define the area; live pixel size is shown.
  • Presets: 1920×1080, 1280×720, or Full (entire monitor).
  • Enter confirms; Esc cancels (keeps the previous region if one existed).

Click Region again later to change the area (when not actively recording).

Live preview

The preview panel shows a throttled live view of the selected source. Preview pauses during recording and when you navigate away or minimize — this saves CPU and GPU.

Video settings

Recording profiles

Profiles bundle container, frame rate, quality, and audio settings. They do not change the encoder — hardware availability is machine-specific.

Built-in profileTypical use
Tutorial (Balanced quality, 30 fps)How-to videos with voice and system audio
Gameplay (High quality, 60 fps)Fast motion, system audio only
Meeting (Standard quality, 30 fps)Calls with mic and system audio
Bug report (Small file, 30 fps)Short demos, smaller files
Quick clip (Low quality, 15 fps, no audio)Silent, small shareable clips
Archive (Maximum quality, 60 fps, lossless audio)MKV + FLAC for best fidelity

Use Save as… to store custom profiles; Delete removes custom ones only. Press F8 to cycle profiles (skipped while recording). Hover profile names in the list for tooltips.

Encoder

RecMode probes encoders at startup with a short trial encode — only working backends are offered. Hardware H.264, HEVC, and AV1 are used when available; software libx264 / x265 / SVT-AV1 are fallbacks.

Format (container)

ContainerNotes
MP4Best for sharing; AAC audio
MKVCrash-safe intermediate when Safe recording is on; Opus or FLAC audio
MOVSimilar to MP4; safe remux supported
WebMAV1 video + Opus only — incompatible combos are blocked before recording

Quality

The slider maps to encoder-specific quality (CRF/CQ/QP) with a perceptual curve. The label shows a tier (Small file / Balanced / High quality / Visually lossless), an estimated MB/min, and the actual encoder value. Quick buttons: Web (15), Balanced (50), Archive (90).

Brightness

Range −100 to +100, applied on the GPU in the capture pipeline. Changes apply live in preview and during an active recording. Effect intensity depends on your GPU driver.

Safe recording

When enabled (default), RecMode writes to a temporary .recording.mkv and remuxes to MP4/MOV on stop without re-encoding. If the app crashes mid-recording, the MKV can often be recovered on next launch.

Audio

Audio is mixed at 48 kHz stereo and encoded according to the container.

System audio

Captures what your PC is playing (WASAPI loopback). Toggle on/off, adjust volume, and watch the level meter. Past ~82% RMS the meter turns caution amber to warn of near-clipping.

Limit to app

When system audio is on, choose one running application to isolate its audio instead of the full system mix. The target must be running when recording starts; if it is not found, system audio is omitted rather than falling back silently to full-system capture.

Microphone

Enable your mic, set volume, and monitor levels the same way as system audio.

Known beta issue: Full-system loopback audio may produce a silent track in some environments. Per-app audio targeting has been verified separately. Check your output with a short test recording if audio is critical.

Webcam overlay

Add a picture-in-picture camera box on top of the recording (and preview):

  1. Enable Show webcam.
  2. Choose the Camera device.
  3. Set Position (corners) and Size (10–50% of frame width).

If no camera is detected, the toggle is disabled. The overlay is composited on the GPU alongside the main capture.

While recording

Floating toolbar

A bottom-centre bar shows the red recording dot, timer, pause, screenshot, live stats, and stop. It is excluded from capture so it never appears in your video.

Pause and resume

Pause with the toolbar button or F10. Paused time is excluded from the output — the final file has no gap where you paused.

Screenshot

Press F11, use the toolbar, or the Record page button. Saves a PNG of the current source to the screenshots folder and copies it to the clipboard. A brief flash confirms the capture.

Click highlights

Enable Settings → Recording → Highlight mouse clicks. During recording, each click draws an accent ripple that is included in the video.

Draw mode

Toggle Draw on the recording toolbar while recording. Ink covers only the captured area (monitor, window, or region — not the wrong screen). Exit with:

  • The Exit draw mode hint bar
  • Esc (global while annotating)
  • F12 (global while annotating)

Right-click clears strokes. The hint bar and toolbar are carved out of the ink surface so you can always reach exit controls.

Health and disk warnings

  • If the encoder falls behind, stats show a warning and the rec dot may turn amber.
  • Low disk space (< 2 GB) warns at start; critically low space stops the recording gracefully.
  • Slow output drives (< 10 MB/s) warn at start.
  • Recording on battery shows an optional warning.

Library

The Library lists files from your recordings and screenshots folders.

  • Videos — newest first; shows codec, resolution, duration, size, and date when indexed.
  • Screenshots — thumbnails for quick browsing.
  • Open — play or view in the default app.
  • Show in folder — open Explorer with the file selected.
  • Delete — send to Recycle Bin.
  • Record again — apply that video's format, frame rate, quality, and audio toggles to the Record screen (capture target is not restored).

Metadata is stored in Data\library.json as recordings finish.

Schedule

Create timed recordings that run while RecMode is open (including from the tray):

  1. Click New schedule and set name, recurrence (Once / Daily / Weekdays / Weekly), start time, and duration.
  2. Optionally bind a Recording profile; otherwise the schedule follows current Record settings for container/fps/quality/audio.
  3. Toggle schedules on or off; edit or delete as needed.

RecMode does not catch up missed runs from while it was fully closed. An active recording is never interrupted by a schedule.

Settings

Appearance

  • Theme: System, Light, or Dark
  • Accent colour (five presets)
  • Navigation layout: Sidebar, Top bar, or Compact

Encoding defaults

Defaults applied when you open Record or start from tray/CLI. Includes video encoder, container, audio format, audio bitrate, and Limit maximum bitrate guardrail.

Output

  • Recordings folder and screenshots folder
  • Filename pattern with tokens {date} {time} {source} — live preview shown in Settings

Recording

  • Countdown before recording
  • Capture mouse cursor
  • Highlight mouse clicks
  • Auto-split large recordings (size threshold)

Hotkeys

Global shortcuts work while any app has focus. Click Change, press the new chord (modifiers + key), Esc to cancel.

Performance

  • Encoder effort — Fast / Balanced / Quality
  • CPU thread cap — for software encoders only
  • Lower encoder process priority — keeps foreground apps responsive

Common workflows

Tutorial with voice

  1. Profile: Tutorial
  2. Source: Screen or Region
  3. Enable Microphone + System audio
  4. Record → Stop → open from Library

Gameplay clip

  1. Profile: Gameplay
  2. Source: Window → pick the game → Follow selected window on
  3. System audio on; mic off
  4. 60 fps hardware encoder if available

Bug report

  1. Profile: Bug report
  2. Region around the affected UI
  3. Enable Draw mode to circle issues while narrating

Repeat a previous recording

  1. Library → Videos → Record again on a past clip
  2. RecMode applies format, fps, quality, and audio from that entry
  3. Re-select capture source if needed, then Record

Overnight scheduled capture

  1. Enable Start with Windows
  2. Schedule → New → set time, duration, and optional profile
  3. Leave RecMode running in the tray

Privacy

  • No telemetry — RecMode does not collect or transmit usage data.
  • Update checks contact your configured feed only when you enable them and click Check now or launch with check enabled.
  • Crash minidumps are opt-in (off by default).
  • All recordings stay on your machine unless you upload them yourself.

Troubleshooting

ProblemThings to try
Encoder missing or failsUpdate GPU drivers; try software encoder; confirm ffmpeg\ffmpeg.exe exists beside the app
Black or empty window captureSome layered/transparent windows capture poorly — try Screen or Region instead
No audio in outputTest per-app audio; verify mic permission; see known full-system audio limitation
File won't play after crashLook for *.recording.mkv — RecMode remuxes orphans on next launch
Hotkey does nothingAnother app may own the chord — remap in Settings; check for collision warning
Can't exit draw modePress F12 or click Exit draw mode; ink area excludes the hint bar by design
Recording stops with disk messageFree space on the output drive; avoid very slow network shares

Beta limitations

RecMode 0.9.x-beta is feature-rich but still in beta. Expect occasional rough edges.

  • Webcam as a primary source tile is not yet available (overlay works today).
  • Hardware encoder matrix not fully verified on all NVIDIA/Intel systems.
  • All Displays compositing best verified on multi-monitor hardware.
  • Animated GIF export is not included — Quick clip profile is a low-fps MP4 substitute.
